# Label Studio iframe 无缝集成文档

## 概述

Label Studio iframe 无缝集成是 SuperInsight 平台的核心功能之一，通过 iframe 和 PostMessage 通信机制，将 Label Studio 标注界面无缝嵌入到前端管理系统中，实现标注与管理的统一体验。
