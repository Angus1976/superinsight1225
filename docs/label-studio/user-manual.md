# Label Studio iframe 集成用户手册

## 概述

Label Studio iframe 集成为 SuperInsight 平台用户提供了无缝的数据标注体验。通过将 Label Studio 标注界面直接嵌入到管理系统中，用户可以在统一的界面中完成项目管理、任务分配和数据标注等全部工作流程。

## 功能特性

### 核心功能
- **无缝集成**: Label Studio 标注界面直接嵌入到 SuperInsight 平台
- **统一认证**: 使用 SuperInsight 账户直接访问标注功能
- **实时同步**: 标注数据自动同步到 SuperInsight 数据库
- **权限控制**: 基于用户角色的精细化权限管理
- **多格式支持**: 支持图像、文本、音频、视频等多种数据类型

### 高级功能
- **离线标注**: 支持网络中断时的离线标注和自动恢复
- **协作标注**: 多用户同时标注同一项目
- **质量控制**: 内置标注质量检查和审核流程
- **数据导出**: 支持多种格式的标注数据导出

## 用户角色和权限

### 管理员 (Admin)
- **项目管理**: 创建、编辑、删除标注项目
- **用户管理**: 分配用户角色和权限
- **数据管理**: 上传数据、配置标注模板
- **质量控制**: 审核标注结果、设置质量标准
- **系统配置**: 配置集成参数和安全策略

### 项目经理 (Project Manager)
- **项目配置**: 配置标注任务和工作流程
- **任务分配**: 将标注任务分配给标注员
- **进度监控**: 查看项目进度和标注统计
- **质量审核**: 审核标注质量和一致性

### 标注员 (Annotator)
- **数据标注**: 执行分配的标注任务
- **进度查看**: 查看个人标注进度
- **质量反馈**: 接收和处理质量反馈

### 审核员 (Reviewer)
- **质量审核**: 审核其他用户的标注结果
- **反馈提供**: 为标注员提供改进建议
- **统计查看**: 查看审核统计和质量报告

## 快速开始

### 1. 访问标注界面

1. 登录 SuperInsight 平台
2. 在左侧导航栏中点击 "数据标注"
3. 选择要标注的项目
4. 点击 "开始标注" 按钮

### 2. 基础标注流程

#### 图像标注示例

1. **选择标注工具**
   - 在工具栏中选择合适的标注工具（矩形框、多边形、点等）
   - 根据标注任务选择对应的标签类别

2. **执行标注**
   - 在图像上绘制标注区域
   - 为标注区域分配正确的标签
   - 添加必要的属性和描述

3. **保存标注**
   - 点击 "保存" 按钮保存当前标注
   - 系统会自动同步数据到后端

4. **提交任务**
   - 完成所有标注后点击 "提交任务"
   - 任务将进入审核流程

#### 文本标注示例

1. **选择文本片段**
   - 用鼠标选择需要标注的文本片段
   - 系统会高亮显示选中的文本

2. **分配标签**
   - 从标签列表中选择合适的标签
   - 为实体添加属性信息

3. **关系标注**
   - 如果需要标注实体间关系，选择两个实体
   - 从关系类型列表中选择合适的关系

4. **验证和保存**
   - 检查标注的准确性
   - 保存并提交标注结果

## 详细功能说明

### 项目管理

#### 创建新项目

1. **基本信息设置**
   ```
   项目名称: 输入项目名称
   项目描述: 详细描述项目目标和要求
   数据类型: 选择标注数据类型（图像/文本/音频/视频）
   标注类型: 选择标注任务类型（分类/检测/分割/NER等）
   ```

2. **标注配置**
   ```
   标签配置: 定义标注标签和类别
   标注模板: 选择或自定义标注界面模板
   质量标准: 设置标注质量要求和评估标准
   工作流程: 配置标注和审核流程
   ```

3. **用户分配**
   ```
   标注员分配: 选择参与标注的用户
   审核员分配: 指定负责审核的用户
   权限设置: 配置用户在项目中的权限
   ```

#### 数据上传和管理

1. **批量上传数据**
   - 支持拖拽上传或选择文件上传
   - 支持 ZIP 压缩包批量上传
   - 自动检测数据格式和类型

2. **数据预处理**
   - 自动调整图像尺寸和格式
   - 文本数据编码转换
   - 数据质量检查和验证

3. **任务分配**
   - 自动或手动将数据分配给标注员
   - 支持负载均衡和优先级设置
   - 重复标注配置用于质量控制

### 标注界面操作

#### 工具栏功能

1. **标注工具**
   - **矩形框**: 用于目标检测和区域标注
   - **多边形**: 用于不规则形状的精确标注
   - **画笔**: 用于像素级分割标注
   - **关键点**: 用于姿态估计和特征点标注
   - **文本选择**: 用于文本实体和关系标注

2. **编辑工具**
   - **选择**: 选择和移动已有标注
   - **删除**: 删除错误的标注
   - **复制**: 复制标注到其他位置
   - **撤销/重做**: 撤销或重做标注操作

3. **视图控制**
   - **缩放**: 放大或缩小查看细节
   - **平移**: 移动视图查看不同区域
   - **全屏**: 切换全屏模式专注标注
   - **网格**: 显示辅助网格线

#### 标签管理

1. **标签分类**
   ```
   主类别: 大的分类类别
   子类别: 细分的标签类型
   属性标签: 描述性属性信息
   关系标签: 实体间的关系类型
   ```

2. **快捷键操作**
   ```
   数字键 1-9: 快速选择常用标签
   Ctrl+Z: 撤销操作
   Ctrl+Y: 重做操作
   Space: 切换到下一个任务
   Enter: 保存当前标注
   ```

3. **标签搜索**
   - 支持标签名称搜索
   - 支持标签分类筛选
   - 最近使用标签快速访问

### 质量控制

#### 标注质量检查

1. **自动检查**
   - 标注完整性检查
   - 标注一致性验证
   - 异常标注检测

2. **人工审核**
   - 审核员逐一检查标注质量
   - 提供修改建议和反馈
   - 批准或拒绝标注结果

3. **质量指标**
   ```
   准确率: 标注正确的比例
   一致性: 多人标注的一致程度
   完整性: 标注覆盖的完整程度
   效率: 标注速度和时间消耗
   ```

#### 审核流程

1. **提交审核**
   - 标注员完成标注后提交审核
   - 系统自动分配给审核员
   - 审核员接收审核通知

2. **审核操作**
   - 查看标注结果和原始数据
   - 检查标注质量和准确性
   - 提供修改建议或直接修正

3. **审核结果**
   - 批准: 标注通过审核，进入最终数据集
   - 拒绝: 标注需要重新修改
   - 修正: 审核员直接修正标注错误

### 数据导出

#### 导出格式

1. **通用格式**

   - 按质量状态筛选

2. **格式设置**
   - 选择导出格式

   - 选择压缩方式

## 常见问题

### 标注相关

**Q: 如何修改已保存的标注？**
A: 选择要修改的标注，使用编辑工具进行修改，然后重新保存。

**Q: 标注时出现卡顿怎么办？**
A: 检查网络连接，刷新页面，或联系管理员检查系统状态。

**Q: 如何快速切换标签？**
A: 使用数字键 1-9 快速选择常用标签，或使用标签搜索功能。

### 权限相关

**Q: 无法访问某个项目怎么办？**
A: 联系项目管理员或系统管理员分配相应权限。

**Q: 标注保存失败怎么办？**
A: 检查网络连接和权限设置，确保有保存权限。

### 数据相关

**Q: 支持哪些数据格式？**
A: 支持常见的图像格式（JPG、PNG、GIF）、文本格式（TXT、JSON）、音频格式（WAV、MP3）等。

**Q: 如何批量上传数据？**
A: 使用 ZIP 压缩包上传，或使用拖拽功能批量选择文件。
