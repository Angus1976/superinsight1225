## 概述

本体专家协作模块是 SuperInsight 平台的核心功能，支持多专家协同构建和维护企业本体。

### 主要功能

- **专家管理**：注册专家、管理专业领域、跟踪贡献
- **模板库**：使用行业模板快速创建本体
- **实时协作**：多人同时编辑，自动冲突检测
- **审批工作流**：多级审批，确保变更质量
- **合规检查**：自动检查中国法规合规性
- **国际化**：支持多语言本体定义

## 快速开始

### 1. 登录系统

使用您的企业账号登录 SuperInsight 平台。

### 2. 进入本体管理

点击左侧导航栏的「本体管理」进入本体专家协作模块。

### 3. 选择或创建本体

- 从模板创建：选择行业模板快速开始
- 从头创建：创建空白本体

### 4. 开始协作

邀请团队成员加入协作会话，开始共同编辑。

## 专家入门

### 注册为专家

1. 进入「专家管理」页面
2. 点击「注册专家」
3. 填写专业信息：
   - 姓名
   - 邮箱
   - 专业领域（可多选）
   - 语言能力
   - 资质证书

### 专业领域

系统支持以下专业领域：

| 领域 | 描述 |
|------|------|
| 金融 | 银行、保险、投资等 |
| 医疗 | 医院、制药、健康等 |
| 制造 | 工厂、供应链、质量等 |
| 政务 | 政府、公共服务等 |
| 法律 | 合同、知识产权等 |
| 教育 | 学校、培训等 |

### 贡献指标

系统自动跟踪您的贡献：

- **贡献数量**：创建/修改的实体和关系数
- **质量评分**：基于审核通过率的质量分
- **认可度**：其他专家的评价

## 模板使用

### 浏览模板

1. 进入「模板库」页面
2. 按行业筛选模板
3. 查看模板详情：
   - 包含的实体类型
   - 关系类型
   - 验证规则

### 实例化模板

1. 选择模板，点击「使用此模板」
2. 输入本体名称
3. 选择自定义选项：
   - 添加额外实体
   - 移除不需要的实体
   - 修改属性
4. 点击「创建」

### 自定义模板

创建本体后，您可以进一步自定义：

1. 添加新实体类型
2. 定义新关系
3. 设置验证规则
4. 添加多语言翻译

## 实时协作

### 创建协作会话

1. 打开本体
2. 点击「开始协作」
3. 邀请团队成员

### 协作编辑

- **锁定机制**：编辑元素时自动锁定，防止冲突
- **实时同步**：所有修改实时同步给其他参与者
- **在线状态**：查看谁在线、谁在编辑什么

### 冲突解决

当发生编辑冲突时：

1. 系统自动检测冲突
2. 显示冲突对比视图
3. 选择解决方式：
   - 接受对方修改
   - 保留我的修改
   - 手动合并

### 版本历史

- 查看所有历史版本
- 比较任意两个版本
- 恢复到历史版本

## 审批工作流

### 提交变更

1. 完成编辑后，点击「提交审批」
2. 填写变更说明
3. 系统自动进行影响分析
4. 提交到审批队列

### 审批流程

```
提交 → 一级审批 → 二级审批 → ... → 完成
         ↓           ↓
       拒绝/修改   拒绝/修改
```

### 审批操作

作为审批人，您可以：

- **通过**：同意变更
- **拒绝**：拒绝变更（需说明原因）
- **请求修改**：要求提交者修改后重新提交

### 影响分析

提交前，系统自动分析：

- 受影响的实体数量
- 受影响的关系数量
- 迁移工作量估算
- 是否需要高级审批

## 最佳实践

### 命名规范

- 使用清晰、描述性的名称
- 遵循行业术语标准
- 保持命名一致性

### 协作礼仪

- 及时释放锁定的元素
- 编辑前查看他人是否在编辑
- 使用评论功能沟通

### 质量保证

- 提交前自检验证规则
- 添加完整的多语言翻译
- 编写清晰的变更说明

### 合规检查

- 定期运行合规检查
- 关注数据分类结果
- 遵循 PIPL 等法规要求

## 常见问题

### Q: 如何处理编辑冲突？

A: 系统会自动检测冲突并提示。您可以选择接受对方修改、保留自己的修改，或手动合并。

### Q: 审批被拒绝怎么办？

A: 查看拒绝原因，修改后重新提交。

### Q: 如何恢复误删的内容？

A: 使用版本历史功能，找到删除前的版本并恢复。

### Q: 如何添加新的专业领域？

A: 联系系统管理员添加新的专业领域。
