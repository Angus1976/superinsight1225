## 🚀 快速导入步骤

### 方法 1: 通过 UI 手动导入（推荐）

#### 任务 1 导入步骤

1. **访问 Label Studio**
   ```
   http://localhost:8080
   ```

2. **登录**
   - 用户名: `admin@example.com`
   - 密码: `admin`

3. **创建项目**
   - 点击 "Create Project"
   - 项目名称: `图文多模态标注任务`
   - 描述: `综合性图文标注，包含目标检测、分类、NER等`

4. **配置标注界面**
   - 在 "Labeling Setup" 页面
   - 点击 "Code" 标签
   - 复制 `task_configs.md` 中 **任务 1** 的 XML 配置
   - 粘贴到编辑器
   - 点击 "Save"

5. **导入数据**
   - 在项目页面点击 "Import"
   - 选择 `comprehensive_task_1.json` 文件
   - 点击 "Import"
   - 等待导入完成

6. **开始标注**
   - 点击任意任务开始标注
   - 按照界面提示完成各项标注
   - 点击 "Submit" 提交

#### 任务 2 导入步骤

重复上述步骤，但使用：
- 项目名称: `多媒体综合标注任务`
- XML 配置: `task_configs.md` 中 **任务 2** 的配置
- 数据文件: `comprehensive_task_2.json`

### 方法 2: 使用 API 导入（高级）

如果你有有效的 API Token，可以使用 Python 脚本：

```python
import requests
import json

LS_URL = "http://localhost:8080"
LS_TOKEN = "your_token_here"  # 从 Account Settings 获取

headers = {
    "Authorization": f"Token {LS_TOKEN}",
    "Content-Type": "application/json"
}

# 创建项目
project_data = {
    "title": "图文多模态标注任务",
    "description": "综合性图文标注",
    "label_config": """<View>...</View>"""  # 从 task_configs.md 复制
}

response = requests.post(
    f"{LS_URL}/api/projects",
    headers=headers,
    json=project_data
)
project_id = response.json()['id']

# 导入任务
with open('comprehensive_task_1.json', 'r') as f:
    tasks = json.load(f)

for task in tasks:
    requests.post(
        f"{LS_URL}/api/projects/{project_id}/tasks",
        headers=headers,
        json=task
    )
```

## 🎯 标注技巧

### 提高效率

1. **使用快捷键**
   - `Ctrl/Cmd + Enter`: 提交
   - `Ctrl/Cmd + S`: 保存草稿
   - `Ctrl/Cmd + Z`: 撤销
   - `Ctrl/Cmd + Y`: 重做

2. **批量操作**
   - 使用过滤器筛选任务
   - 批量分配任务
   - 批量导出结果

3. **模板复用**
   - 保存常用的标注模板
   - 使用预标注加速流程

### 质量保证

1. **标注前**
   - 仔细阅读标注指南
   - 理解每个标签的含义
   - 熟悉标注界面

2. **标注中**
   - 保持一致性
   - 遇到疑问及时记录
   - 定期保存进度

3. **标注后**
   - 检查是否有遗漏
   - 验证标注的准确性
   - 提交前再次审核

## 📤 导出结果

### 导出格式

1. **JSON** - 完整的标注数据
   ```json
   {
     "id": 1,
     "data": {...},
     "annotations": [{
       "result": [...]
     }]
   }
   ```

2. **CSV** - 表格格式，适合数据分析

3. **COCO** - 目标检测标准格式

4. **YOLO** - YOLO 模型训练格式

### 导出步骤

1. 在项目页面点击 "Export"
2. 选择导出格式
3. 点击 "Export" 下载文件

## 🔧 常见问题

### Q1: 图片/视频加载失败？
**A**: 检查 URL 是否可访问，或使用本地文件

### Q2: 标注保存失败？
**A**: 检查网络连接，确认 Label Studio 服务正常运行

### Q3: 如何修改已提交的标注？
**A**: 在任务列表中点击任务，重新编辑后提交

### Q4: 如何邀请团队成员？
**A**: 在 Settings → Members 中添加成员邮箱

### Q5: 如何设置标注审核？
**A**: 在 Settings → Quality 中配置审核流程
