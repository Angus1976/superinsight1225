# 数据生命周期管理系统 - 手动测试清单

**版本**: 1.0  
**日期**: 2026-02-04  
**测试范围**: 数据生命周期管理 UI 工作流  
**验证需求**: Requirements 11.5, 19.1, 19.3, 19.5, 19.6

---

## 测试目标

本测试清单用于验证数据生命周期管理系统的 UI 工作流是否正确运行，包括：

1. **完整数据生命周期流程** - 从上传到增强的端到端流程
2. **状态转换** - 所有 13 个状态之间的转换
3. **权限控制** - UI 中的权限强制执行
4. **国际化** - 中英文切换功能
5. **硬编码字符串检查** - 验证无硬编码文本

---

### 2. 状态转换测试

#### 2.1 有效状态转换

**测试所有有效的状态转换路径**:

| 当前状态 | 目标状态 | 操作 | 预期结果 |
|---------|---------|------|---------|
| raw | structured | 解析文档 | ✓ 转换成功 |
| structured | temp_stored | 存储到临时表 | ✓ 转换成功 |
| temp_stored | under_review | 提交审核 | ✓ 转换成功 |
| under_review | approved | 批准 | ✓ 转换成功 |
| under_review | rejected | 拒绝 | ✓ 转换成功 |
| approved | in_sample_library | 转移到样本库 | ✓ 转换成功 |
| in_sample_library | annotation_pending | 创建标注任务 | ✓ 转换成功 |
| annotation_pending | annotating | 开始标注 | ✓ 转换成功 |
| annotating | annotated | 完成标注 | ✓ 转换成功 |
| annotated | enhancing | 开始增强 | ✓ 转换成功 |
| enhancing | enhanced | 完成增强 | ✓ 转换成功 |
| enhanced | in_sample_library | 添加到样本库 | ✓ 转换成功 |
| enhanced | archived | 归档 | ✓ 转换成功 |

**测试步骤**:
1. 对于每个状态转换，执行相应的操作
2. 验证状态正确更新

**预期结果**:
