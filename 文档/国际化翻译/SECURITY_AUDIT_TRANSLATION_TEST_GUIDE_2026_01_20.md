## 清除浏览器缓存

如果看到旧的翻译，请清除缓存：

### Chrome
```
1. Ctrl+Shift+Delete
2. 选择 "所有时间"
3. 勾选 "Cookie 和其他网站数据"
4. 点击 "清除数据"
```

### Firefox
```
1. Ctrl+Shift+Delete
2. 选择 "所有"
3. 点击 "清除"
```

### Safari
```
1. Develop > Empty Web Caches
2. 或者: Cmd+Option+E
```

### 硬刷新
```
Windows/Linux: Ctrl+F5
Mac: Cmd+Shift+R
```

## 故障排除

### 问题: 看到翻译键而不是翻译文本
**解决方案**:
1. 清除浏览器缓存
2. 硬刷新页面 (Ctrl+F5 或 Cmd+Shift+R)
3. 检查浏览器控制台是否有错误

### 问题: 看到英文而不是中文
**解决方案**:
1. 检查语言设置是否为中文
2. 切换到其他语言再切换回中文
3. 清除缓存并刷新

### 问题: 参数化翻译不显示数字
**解决方案**:
1. 检查浏览器控制台是否有错误
2. 确认有未解决的安全事件
3. 刷新页面

## 浏览器控制台检查

打开浏览器开发者工具 (F12) 并检查:

1. **Console 标签**:
   - 是否有红色错误信息
   - 是否有翻译相关的警告

2. **Network 标签**:
   - 翻译文件是否正确加载
   - 是否有 404 错误

3. **Application 标签**:
   - 检查 localStorage 中的语言设置
   - 检查 i18n 配置
