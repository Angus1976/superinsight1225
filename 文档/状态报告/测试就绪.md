## 🆘 故障排除

### 如果后端无响应
```bash
pkill -f simple_app.py
python3 simple_app.py
```

### 如果前端无响应
```bash
pkill -f "npm run dev"
cd frontend && npm run dev
```

### 如果登录失败
1. 检查浏览器控制台是否有错误
2. 清除浏览器缓存
3. 重新加载页面

### 如果应用加载缓慢
1. 检查网络连接
2. 检查浏览器开发者工具中的 Network 标签
3. 检查后端日志
