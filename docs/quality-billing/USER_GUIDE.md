# 质量计费闭环系统 - 用户指南

## 1. 系统简介

质量计费闭环系统帮助您管理数据标注质量、跟踪团队绩效、并实现基于质量的精准计费。本指南将帮助您快速上手使用系统的各项功能。

## 2. 快速开始

### 2.1 登录系统

1. 打开浏览器，访问系统地址
2. 输入您的用户名和密码
3. 点击「登录」按钮

### 2.2 界面概览

登录后您将看到主控制台，包含以下区域：

- **顶部导航栏**: 切换不同功能模块
- **左侧菜单**: 当前模块的子功能
- **主内容区**: 显示具体内容和操作
- **右侧面板**: 快捷操作和通知

## 3. 工单管理

### 3.1 查看工单列表

1. 点击顶部导航栏的「工单管理」
2. 使用筛选器按状态、优先级筛选工单
3. 点击工单编号查看详情

### 3.2 工单状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| 待处理 | 🟡 | 新创建，等待分配 |
| 处理中 | 🔵 | 已分配，正在处理 |
| 待验证 | 🟣 | 处理完成，等待质量验证 |
| 已完成 | 🟢 | 验证通过，工单关闭 |
| 已取消 | ⚫ | 工单已取消 |

### 3.3 处理工单

1. 在工单详情页，点击「开始处理」
2. 根据问题描述进行修复操作
3. 填写修复说明
4. 点击「提交修复」
5. 等待系统自动验证质量

### 3.4 工单优先级

- **紧急**: 需在 2 小时内响应，4 小时内解决
- **高**: 需在 4 小时内响应，8 小时内解决
- **中**: 需在 8 小时内响应，24 小时内解决
- **低**: 需在 24 小时内响应，72 小时内解决

## 4. 绩效查看

### 4.1 查看个人绩效

1. 点击右上角头像，选择「我的绩效」
2. 查看各维度评分和综合得分
3. 查看绩效趋势图

### 4.2 绩效维度说明

| 维度 | 权重 | 说明 |
|------|------|------|
| 质量分 | 40% | 标注和修复的质量得分 |
| 效率分 | 30% | 完成任务的速度和及时性 |
| 合规分 | 20% | 遵守规范和流程的程度 |
| 改进分 | 10% | 质量持续改进的表现 |

### 4.3 绩效等级

| 等级 | 综合分 | 说明 |
|------|--------|------|
| A+ | ≥ 0.95 | 卓越，持续超越期望 |
| A | ≥ 0.90 | 优秀，稳定高质量产出 |
| B | ≥ 0.80 | 良好，符合预期标准 |
| C | ≥ 0.70 | 合格，达到基本要求 |
| D | ≥ 0.60 | 待改进，需加强培训 |
| E | < 0.60 | 不合格，需立即改进 |

### 4.4 查看团队排名

1. 点击「绩效管理」→「团队排名」
2. 选择评估周期（日/周/月）
3. 查看排名和对比数据

## 5. 质量监控

### 5.1 质量仪表盘

质量仪表盘展示关键质量指标：

- **平均质量分**: 整体质量水平
- **质量等级分布**: 各等级占比
- **质量趋势**: 近期质量变化
- **问题热点**: 常见质量问题

### 5.2 质量报告

1. 点击「质量监控」→「质量报告」
2. 选择报告周期
3. 点击「生成报告」
4. 下载或在线查看报告

### 5.3 质量告警

当出现以下情况时，系统会发送告警：

- 质量分低于阈值
- 质量下降幅度过大
- SLA 即将违规
- 异常模式检测

## 6. 计费管理

### 6.1 查看账单

1. 点击「计费管理」→「账单列表」
2. 选择计费周期
3. 查看账单详情

### 6.2 账单说明

账单包含以下信息：

- **计费周期**: 账单覆盖的时间范围
- **明细项**: 各质量等级的使用量和金额
- **质量证书**: 质量评估认证信息
- **总金额**: 应付款项总额

### 6.3 计费规则

| 质量等级 | Token 单价 | 说明 |
|----------|------------|------|
| Platinum | ¥0.15/千 | 最高质量，评分 ≥ 0.90 |
| Gold | ¥0.12/千 | 高质量，评分 ≥ 0.75 |
| Silver | ¥0.08/千 | 标准质量，评分 ≥ 0.60 |
| Bronze | ¥0.05/千 | 基础质量，评分 < 0.60 |

### 6.4 查看质量证书

每份账单都附带质量证书，证明服务质量：

1. 在账单详情页，点击「查看质量证书」
2. 查看评估维度和得分
3. 下载 PDF 格式证书

## 9. 常见问题

### Q1: 如何提高绩效分数？

**A**: 提高绩效的关键：
- 确保标注和修复的质量
- 及时处理分配的工单
- 遵守工作规范和流程
- 持续学习和改进

### Q2: 质量评分是如何计算的？

**A**: 系统使用 Ragas 评估框架，综合以下维度：
- 忠实度 (Faithfulness): 回答与上下文的一致性
- 相关性 (Relevancy): 回答与问题的相关程度
- 精确度 (Precision): 上下文使用的精确程度
- 召回率 (Recall): 关键信息的覆盖程度

### Q3: 工单 SLA 快到期了怎么办？

**A**:
1. 立即处理或请求协助
2. 如无法按时完成，及时说明原因
3. 系统会自动升级紧急工单

### Q4: 账单有疑问怎么处理？

**A**:
1. 在账单详情页点击「申诉」
2. 填写疑问说明
3. 等待财务团队审核处理

### Q5: 如何查看培训资料？

**A**:
1. 点击「帮助中心」→「培训资料」
2. 选择需要学习的主题
3. 观看视频或阅读文档
