🔧 常用命令
═══════════════════════════════════════════════════════════════════════════

启动服务:
  $ docker compose up -d

停止服务:
  $ docker compose down

查看服务状态:
  $ docker compose ps

查看实时日志:
  $ docker compose logs -f superinsight-api

进入数据库:
  $ docker compose exec postgres psql -U superinsight -d superinsight

重置数据库:
  $ docker compose down -v
  $ docker compose up -d
  $ docker compose exec superinsight-api python scripts/seed_demo_data.py
