## 🎨 专业商务配色方案

### 主配色：深蓝专业渐变

针对**数据标注AI公司**的专业商务形象设计：

| 颜色名称 | 色值 | 应用位置 | 象征意义 |
|---------|------|----------|----------|
| **深蓝** | #0B2847 | 立体左侧面 | 数据专业、深度、可信 |
| **品牌蓝** | #1890FF | 立体上表面 | 科技、创新、数据 |
| **天蓝** | #40A9FF | 立体顶部高光 | 明亮、清晰、智能 |
| **科技蓝紫** | #597EF7 | 立体右侧面 | AI、智能、前沿 |
| **洞察红** | #FF4D4F | i的圆点 | SuperInsight核心 ⭐ |
| **数据绿** | #52C41A | 数据点、箭头 | 数据友好、成功 |

### 为什么选择深蓝渐变？

**专业商务考虑**：
1. ✅ **深蓝 #0B2847** - 专业、稳重、商务感强
2. ✅ **品牌蓝 #1890FF** - 科技、数据、信任感
3. ✅ **科技紫 #597EF7** - AI、智能、创新感
4. ✅ 避免过于鲜艳的颜色
5. ✅ 符合B2B企业形象
6. ✅ 适合数据标注行业特征

**与竞品差异**：
- ❌ 不用常见的蓝绿渐变（太普通）
- ✅ 采用深蓝到蓝紫渐变（更专业）
- ✅ 立体效果增强商务感

## 🎨 设计元素详解

### 核心元素：几何立体"Wi"

基于手绘草图的3D立体设计：

| 元素 | 结构 | 作用 | 象征意义 |
|------|------|------|----------|
| **W柱1** | 3面立体柱 | W的左侧 | 数据采集起点 |
| **W柱2** | 3面立体柱（最短） | W的谷底 | 数据清洗处理 |
| **W柱3** | 3面立体柱 | W的右侧 | 数据标注上升 |
| **i柱** | 3面立体柱（最高） | i的主体 | AI智能训练 |
| **红点** | ● | i的点 | SuperInsight洞察 ⭐ |

### 3D立体效果

每个柱体由**3个面**组成：

1. **上表面（Top Face）** - 浅蓝 #40A9FF → 品牌蓝 #1890FF
   - 渐变从上到下
   - 明亮，有光泽

2. **左侧面（Left Face）** - 深蓝 #0B2847 → 中蓝 #1677D9
   - 较暗，增强立体感
   - 专业商务感

3. **右侧面（Right Face）** - 中蓝 #1677D9 → 科技紫 #597EF7
   - 科技感
   - AI智能象征

### 视觉层次

```
视觉深度层次（从深到浅）：

第1层：深蓝 #0B2847（最深，左侧面）
  ↓ 专业商务感
第2层：中蓝 #1677D9（过渡色）
  ↓ 科技数据感
第3层：品牌蓝 #1890FF（主色）
  ↓ 品牌识别
第4层：天蓝 #40A9FF（高光）
  ↓ 明亮智能
第5层：科技紫 #597EF7（AI感）
  ↓ 智能创新

强调色：
- 红点 #FF4D4F（洞察核心）⭐
- 绿色 #52C41A（数据友好）
```

## 🚀 快速使用指南

### 1. HTML 中使用

```html
<!-- 网站 Logo（静态版，推荐） -->
<img src="/logos/logo-full-static.svg" alt="问视间 WinsAI" height="80">

<!-- 启动页（动画版） -->
<img src="/logos/logo-full.svg" alt="WinsAI SuperInsight" height="80">

<!-- Favicon -->
<link rel="icon" type="image/svg+xml" href="/logos/logo-favicon.svg">

<!-- PWA -->
<link rel="apple-touch-icon" href="/logos/logo-square-256.svg">
<link rel="manifest" href="/manifest.json">
```

### 2. React/TypeScript 中使用

```tsx
// Logo 组件
interface LogoProps {
  variant?: 'full' | 'full-static' | 'icon' | 'simple';
  animated?: boolean;
  size?: number;
}

export function WinsAILogo({ 
  variant = 'full-static', 
  animated = false,
  size = 60 
}: LogoProps) {
  const logoSrc = animated 
    ? '/logos/logo-full.svg'
    : `/logos/logo-${variant}.svg`;

  return (
    <img 
      src={logoSrc} 
      alt="问视间 WinsAI - 数据友好·智赢未来" 
      height={size}
      className="winsai-logo"
    />
  );
}

// 使用示例
<WinsAILogo variant="full-static" size={60} /> {/* Header */}
<WinsAILogo variant="icon-64" size={48} /> {/* 侧边栏 */}
<WinsAILogo animated={true} /> {/* 启动页 */}
```

### 3. CSS 中使用

```css
/* 背景图 */
.header-logo {
  background-image: url('/logos/logo-horizontal.svg');
  background-size: contain;
  background-repeat: no-repeat;
  width: 360px;
  height: 72px;
}

/* 深色模式适配 */
@media (prefers-color-scheme: dark) {
  .winsai-logo {
    filter: brightness(1.1);
  }
}
```

## 📏 设计规范

### 最小尺寸

| 版本 | 最小宽度 | 说明 |
|------|----------|------|
| 完整版 | 180px | 保持立体清晰 |
| 图标版 | 32px | 立体结构清晰 |
| 简化版 | 24px | 识别度尚可 |
| Favicon | 16px | 红点是关键 |

### 安全区域

- Logo 周围保留 **Logo 高度 20%** 的留白
- 给 Logo 足够的呼吸空间
- 确保红点和立体效果清晰

### 颜色使用

**适用背景**：
- ✅ 白色背景（最佳）⭐
- ✅ 浅色背景（很好）
- ✅ 深色背景（需注意对比度）
- ✅ 浅灰背景 #F5F5F5（推荐）

**禁止操作** ❌：
- ❌ 不要改变颜色（特别是红点和深蓝）
- ❌ 不要拉伸变形（破坏立体效果）
- ❌ 不要旋转 Logo
- ❌ 不要分离元素
- ❌ 不要移除红点（核心辨识）
- ❌ 不要改变立体面的颜色关系

**推荐操作** ✅：
- ✅ 保持原始宽高比
- ✅ 等比例缩放
- ✅ 使用纯色背景
- ✅ 保持立体清晰
- ✅ 确保红点醒目
