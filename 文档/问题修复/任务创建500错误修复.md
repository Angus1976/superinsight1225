# 任务创建 500 错误修复

**日期**: 2026-02-02  
**状态**: ✅ 已解决

## 问题

创建任务时返回 500 错误，无法成功创建。

## 根本原因

代码与数据库字段类型不匹配，共 7 个问题：

1. **表单验证不完整** - `validateFields()` 只验证可见字段
2. **Mock UUID 格式错误** - 使用简单字符串而非 UUID 格式
3. **异步函数问题** - BackgroundTasks 不支持 async 函数
4. **created_by 类型** - 代码用 String，数据库是 UUID
5. **SimpleUser 属性错误** - 使用 `user_id` 但实际是 `id`
6. **title 字段缺失** - 数据库有 NOT NULL 的 title 字段，模型中没有
7. **assignee_id 类型** - UUID 类型需要正确转换

## 解决方案

### 1. 表单验证
```typescript
// frontend/src/pages/Tasks/TaskCreateModal.tsx
await form.validateFields(['name', 'annotation_type', 'priority', 'assignee_id', 'due_date', 'total_items', 'tags', 'data_source']);
```

### 2. Mock UUID 格式
```typescript
// frontend/src/services/mockData.ts
{ id: 'b0000001-0000-4000-8000-000000000001', ... }
```

### 3. 后端异步处理
```python
# src/api/tasks.py
def _sync_task_to_label_studio(...):  # 改为同步函数
    with ThreadPoolExecutor() as executor:
        future = executor.submit(asyncio.run, async_function())
```

### 4. 数据库类型对齐
```python
# src/database/models.py
created_by: Mapped[UUID] = mapped_column(UUID(as_uuid=True), nullable=False)

# src/api/tasks.py
created_by=UUID(current_user.id)
```

### 5. SimpleUser 属性
```python
# src/api/label_studio_api.py
user_id = current_user.id  # 不是 current_user.user_id
```

### 6. title 字段
```python
# src/database/models.py
title: Mapped[str] = mapped_column(String(255), nullable=False, default="Untitled Task")

# src/api/tasks.py
task = TaskModel(
    title=request.name,
    name=request.name,
    ...
)
```
