## 🚀 快速开始

### 1. 自动化设置 (推荐)
```bash
chmod +x fullstack_setup.sh
./fullstack_setup.sh
```

### 2. 启动服务

**后端:**
```bash
source venv/bin/activate
python -m uvicorn src.app:app --host 0.0.0.0 --port 8000 --reload
```

**前端:**
```bash
cd frontend
npm run dev
```

### 3. 访问应用
- 前端: http://localhost:5173
- 后端 API: http://localhost:8000
- API 文档: http://localhost:8000/docs
