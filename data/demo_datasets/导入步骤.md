## 导入步骤

### 方法 1: 通过 UI 导入（推荐）

1. **在当前项目页面，点击右上角的 "Import" 按钮**

2. **选择导入方式**
   - 点击 "Upload Files" 
   - 或直接拖拽 JSON 文件到页面

3. **选择数据文件**
   
   根据您的项目类型，选择对应的文件：
   
   **文本类项目**:
   - `text_classification.json` - 文本分类（10条）
   - `ner_dataset.json` - 命名实体识别（10条）
   - `qa_dataset.json` - 问答系统（5条）
   
   **图像类项目**:
   - `image_classification.json` - 图像分类（10张）
   - `object_detection.json` - 目标检测（5张）
   
   **综合任务**:
   - `comprehensive_task_1.json` - 图文多模态（5个样本）⭐ 推荐
   - `comprehensive_task_2.json` - 多媒体综合（5个样本）⭐ 推荐

4. **确认导入**
   - 点击 "Import" 按钮
   - 等待导入完成
   - 页面会显示导入的任务数量

5. **开始标注**
   - 导入成功后，任务列表会显示所有任务
   - 点击任意任务开始标注

## 方法 2: 使用示例数据（最简单）

如果您只是想快速测试，可以手动创建一个简单任务：

1. 在项目页面点击 "Import"
2. 选择 "Add Task" 或 "Paste JSON"
3. 粘贴以下示例数据：

\`\`\`

4. 点击 "Add Task"
5. 任务创建成功，可以开始标注

## 常见问题

### Q: 找不到 Import 按钮？
**A**: Import 按钮通常在项目页面的右上角，或者在任务列表页面的顶部工具栏

### Q: 导入后还是没有任务？
**A**: 
1. 检查 JSON 文件格式是否正确
2. 刷新页面
3. 检查是否有错误提示

### Q: JSON 格式错误？
**A**: 确保 JSON 文件：
- 使用 UTF-8 编码
- 是有效的 JSON 数组格式 `[{...}, {...}]`
- 字段名与标注配置中的变量名匹配（如 `$text`, `$image`）

### Q: 图片无法加载？
**A**: 
- 确保图片 URL 可以从浏览器访问
- 检查是否有 CORS 限制
- 可以使用本地文件或公开的图片 URL
