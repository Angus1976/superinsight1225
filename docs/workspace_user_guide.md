## 概述

### 什么是工作空间？

工作空间（Workspace）是 Label Studio Enterprise 中用于组织和管理数据标注项目的核心概念。通过工作空间，您可以：

- **组织项目**：将相关的标注项目归类到同一工作空间
- **团队协作**：邀请团队成员参与标注工作
- **权限管理**：通过角色分配控制成员的访问权限
- **数据隔离**：确保不同团队或项目之间的数据安全

### 核心功能

| 功能 | 描述 |
|------|------|
| 工作空间 CRUD | 创建、查看、编辑、删除工作空间 |
| 成员管理 | 添加、移除成员，分配角色 |
| 项目关联 | 将 Label Studio 项目关联到工作空间 |
| 权限控制 | 基于角色的细粒度访问控制 |

## 快速开始

### 第一步：创建工作空间

1. 登录 SuperInsight 平台
2. 点击左侧菜单中的「Label Studio 工作空间」
3. 点击右上角的「创建工作空间」按钮
4. 填写工作空间信息：
   - **名称**：为工作空间取一个有意义的名称（如：产品图像标注）
   - **描述**：简要描述工作空间的用途（可选）
5. 点击「创建」完成

### 第二步：添加团队成员

1. 在工作空间列表中，点击刚创建的工作空间
2. 切换到「成员」标签页
3. 点击「添加成员」按钮
4. 搜索并选择要添加的用户
5. 选择合适的角色：
   - **管理员**：协助管理工作空间
   - **经理**：管理项目和分配任务
   - **审核员**：审核标注结果
   - **标注员**：执行标注任务
6. 点击「确认」添加成员

### 第三步：关联项目

1. 在工作空间详情页，切换到「项目」标签页
2. 点击「关联项目」按钮
3. 输入 Label Studio 项目 ID
4. 点击「关联」完成

现在您的团队成员就可以在此工作空间中协作进行数据标注了！

## 工作空间管理

### 创建工作空间

**操作步骤**：

1. 导航至「Label Studio 工作空间」页面
2. 点击「创建工作空间」
3. 填写必填信息：
   - 名称（必填，1-255 字符，不可重复）
   - 描述（可选，最多 2000 字符）
4. 点击「创建」

**注意事项**：
- 工作空间名称在系统中必须唯一
- 创建者自动成为工作空间的所有者（Owner）

### 编辑工作空间

**操作步骤**：

1. 在工作空间列表中找到目标工作空间
2. 点击工作空间名称进入详情页
3. 点击「编辑」按钮
4. 修改需要更新的信息
5. 点击「保存」

**权限要求**：需要 `workspace:edit` 权限（Owner 或 Admin）

### 删除工作空间

**操作步骤**：

1. 在工作空间详情页
2. 点击「更多操作」→「删除工作空间」
3. 在确认对话框中输入工作空间名称
4. 点击「确认删除」

**注意事项**：
- 只有工作空间所有者（Owner）可以删除
- 如果工作空间包含关联项目，需要先取消所有项目关联
- 删除操作为软删除，数据保留 30 天后彻底清除

### 停用/启用工作空间

**操作步骤**：

1. 在工作空间详情页
2. 点击「更多操作」→「停用工作空间」或「启用工作空间」

停用后的工作空间：
- 不会在默认列表中显示
- 成员无法访问其中的项目
- 所有数据保留，可随时重新启用

## 成员管理

### 添加成员

**操作步骤**：

1. 进入工作空间详情页
2. 切换到「成员」标签页
3. 点击「添加成员」
4. 搜索用户（支持用户名、邮箱搜索）
5. 选择要分配的角色
6. 点击「添加」

**权限要求**：需要 `workspace:manage_members` 权限

### 修改成员角色

**操作步骤**：

1. 在成员列表中找到目标成员
2. 点击角色下拉框
3. 选择新角色
4. 确认修改

**角色修改规则**：
- Owner 可以修改任何成员的角色
- Admin 可以修改除 Owner 外的成员角色
- 不能将最后一个 Owner 降级为其他角色

### 移除成员

**操作步骤**：

1. 在成员列表中找到目标成员
2. 点击「移除」按钮
3. 确认移除操作

**注意事项**：
- 无法移除最后一个 Owner
- 被移除的成员将立即失去工作空间访问权限

### 成员角色说明

| 角色 | 图标颜色 | 适用场景 |
|------|----------|----------|
| Owner（所有者） | 金色 | 工作空间创建者，拥有完全控制权 |
| Admin（管理员） | 紫色 | 协助管理工作空间和成员 |
| Manager（经理） | 蓝色 | 项目负责人，管理项目和任务分配 |
| Reviewer（审核员） | 绿色 | 审核标注质量 |
| Annotator（标注员） | 灰色 | 执行标注任务 |

## 项目管理

### 关联项目

将现有的 Label Studio 项目关联到工作空间，便于统一管理。

**操作步骤**：

1. 进入工作空间详情页
2. 切换到「项目」标签页
3. 点击「关联项目」
4. 输入 Label Studio 项目 ID
5. 点击「关联」

**权限要求**：需要 `project:create` 权限

**注意事项**：
- 一个项目只能关联到一个工作空间
- 已关联到其他工作空间的项目需要先取消关联

### 查看项目列表

**操作步骤**：

1. 进入工作空间详情页
2. 切换到「项目」标签页
3. 可以看到所有关联的项目列表

列表显示信息：
- 项目名称
- 项目描述
- 关联时间
- 快捷操作

### 在 Label Studio 中打开项目

**操作步骤**：

1. 在项目列表中找到目标项目
2. 点击「在 Label Studio 中打开」图标
3. 将在新标签页中打开 Label Studio 项目页面

### 取消项目关联

**操作步骤**：

1. 在项目列表中找到目标项目
2. 点击「移除」按钮
3. 确认取消关联

**权限要求**：需要 `project:delete` 权限

**注意事项**：
- 取消关联不会删除项目本身
- 取消关联后，项目可以被关联到其他工作空间

## 角色与权限

### 权限矩阵

| 功能 | Owner | Admin | Manager | Reviewer | Annotator |
|------|:-----:|:-----:|:-------:|:--------:|:---------:|
| 查看工作空间 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 编辑工作空间 | ✓ | ✓ | - | - | - |
| 删除工作空间 | ✓ | - | - | - | - |
| 管理成员 | ✓ | ✓ | - | - | - |
| 查看项目 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 创建/关联项目 | ✓ | ✓ | ✓ | - | - |
| 编辑项目 | ✓ | ✓ | ✓ | - | - |
| 删除/取消关联项目 | ✓ | ✓ | - | - | - |
| 管理项目成员 | ✓ | ✓ | ✓ | - | - |
| 查看任务 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 执行标注 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 审核标注 | ✓ | ✓ | ✓ | ✓ | - |
| 分配任务 | ✓ | ✓ | ✓ | - | - |
| 导出数据 | ✓ | ✓ | ✓ | ✓ | - |
| 导入数据 | ✓ | ✓ | - | - | - |

### 角色选择建议

**Owner（所有者）**
- 项目总负责人
- 需要完全控制权的管理人员
- 每个工作空间至少保留一个 Owner

**Admin（管理员）**
- 协助 Owner 管理工作空间
- 负责成员管理和日常运维
- 不需要删除工作空间权限的管理人员

**Manager（经理）**
- 项目经理或团队负责人
- 负责具体项目的执行和管理
- 需要分配任务但不需要管理成员

**Reviewer（审核员）**
- 质量审核人员
- 负责审核标注结果
- 需要导出数据进行质量分析

**Annotator（标注员）**
- 执行具体标注工作的人员
- 仅需要查看和标注权限
- 不参与项目管理

## 常见问题

### Q1: 如何转让工作空间所有权？

**A**:
1. 先将目标用户添加为成员
2. 将其角色修改为 Owner
3. 原 Owner 可以保留 Owner 角色，或降级为其他角色

### Q2: 为什么无法删除工作空间？

**A**: 可能原因：
- 您不是工作空间的 Owner
- 工作空间中还有关联的项目（需要先取消所有项目关联）

### Q3: 成员被移除后还能访问之前的数据吗？

**A**: 不能。成员被移除后立即失去工作空间及其所有项目的访问权限。

### Q4: 一个用户可以加入多少个工作空间？

**A**: 没有限制。用户可以加入任意数量的工作空间，并在不同工作空间中拥有不同的角色。

### Q5: 如何批量添加成员？

**A**: 目前仅支持逐个添加成员。批量导入功能将在后续版本中提供。

### Q6: 工作空间的数据存储在哪里？

**A**: 工作空间的配置数据存储在 SuperInsight 平台数据库中，标注数据存储在 Label Studio 中。

### Q7: 删除工作空间后数据还能恢复吗？

**A**: 软删除后的数据保留 30 天，在此期间可以联系管理员恢复。超过 30 天后数据将被永久删除。

### Q8: 如何查看我的权限？

**A**: 在工作空间详情页右上角可以看到您当前的角色和权限列表。
