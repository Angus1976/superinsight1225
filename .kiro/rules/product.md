# SuperInsight AI 数据治理与标注平台

**类型**: 项目信息  
**优先级**: MEDIUM  
**加载方式**: 手动加载（按需引用）

---

## 产品定位

企业级 AI 数据治理与智能标注平台，深度整合"理采存管用"方法论，全面升级支持 LLM 和 GenAI 应用场景。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 安全数据提取 | 只读权限提取多源数据（数据库、文件、API） |
| AI 预标注 | 集成多种 LLM 模型智能预标注 |
| 人机协作 | 支持业务专家与技术专家协同标注 |
| 质量管理 | 基于 Ragas 框架的语义质量评估 |
| 计费结算 | 精确的工时和计件统计 |
| 安全合规 | 企业级安全控制和审计系统 |
| 多种部署 | 支持云托管、私有部署、混合云 |

---

## 技术基础

基于 Label Studio 标注引擎，配备实时安全监控、自动数据脱敏、多租户架构、智能质量评估系统。

---

## 目标用户

需要安全、合规、可扩展的 AI 数据标注和治理解决方案的企业客户，特别是处理敏感数据或需要严格审计追踪的场景。
