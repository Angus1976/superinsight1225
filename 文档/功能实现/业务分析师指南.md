# 业务分析师使用指南

## 概述

本指南专为业务分析师设计，帮助您充分利用 SuperInsight 的业务逻辑提炼与智能化功能，从标注数据中发现有价值的业务洞察，支持数据驱动的决策制定。

## 快速入门

### 1. 访问业务逻辑仪表板

登录 SuperInsight 平台后，导航到 **业务逻辑** 模块：

1. 点击左侧菜单的 **"业务分析"**

## 核心功能使用

### 1. 业务模式分析

#### 1.1 启动模式分析

1. 在仪表板中点击 **"开始分析"** 按钮
2. 选择分析类型：
   - ✅ **情感关联分析**: 发现情感与关键词的关联
   - ✅ **关键词共现分析**: 识别关键词组合模式
   - ✅ **时间趋势分析**: 分析数据随时间的变化
   - ✅ **用户行为分析**: 了解用户标注行为模式

3. 设置分析参数：
   ```
   置信度阈值: [0.7] ────────────── (推荐: 0.6-0.8)
   支持度阈值: [5]   ────────────── (推荐: 3-10)
   时间范围:   [最近30天] ──────── (可自定义)
   ```

4. 点击 **"执行分析"**

#### 1.2 解读分析结果

**情感关联分析结果示例**：
```

│ 置信度: 0.85  支持度: 45  提升度: 1.67                  │
│ 关键词: "excellent", "quality", "satisfied", "perfect"  │
│ 业务含义: 用户提到产品质量时，85%的概率表达正面情感      │
```

**关键指标解释**：
- **置信度**: 规则的可靠性，越高越可信
- **支持度**: 支持该规则的样本数量，反映普遍性
- **提升度**: 相对于随机情况的提升倍数，>1表示正相关

## 最佳实践

### 1. 数据质量保证

#### 1.1 分析前检查
- ✅ 确保数据量充足 (建议 >100 条)
- ✅ 检查数据完整性和一致性
- ✅ 验证时间范围的合理性
- ✅ 确认标注质量符合要求

#### 1.2 参数设置建议
```
数据量级        置信度阈值    支持度阈值
─────────────────────────────────────
< 500 条        0.6          3
500-2000 条     0.7          5
2000-5000 条    0.75         8
> 5000 条       0.8          10
```

### 2. 结果解读技巧

#### 2.1 置信度解读
- **0.9-1.0**: 极高可信度，可直接应用
- **0.8-0.9**: 高可信度，建议应用
- **0.7-0.8**: 中等可信度，需要验证
- **0.6-0.7**: 较低可信度，谨慎使用
- **< 0.6**: 低可信度，不建议使用

#### 2.2 支持度解读
- **支持度高**: 规律普遍存在，适用范围广
- **支持度低**: 可能是特殊情况或小众模式
- **建议**: 结合业务场景判断支持度的重要性

#### 2.3 提升度解读
- **> 2.0**: 强正相关，值得重点关注
- **1.5-2.0**: 中等正相关，有应用价值
- **1.0-1.5**: 弱正相关，需结合其他因素
- **< 1.0**: 负相关，可能存在抑制关系

### 3. 业务应用策略

#### 3.1 规则应用优先级
1. **高置信度 + 高支持度**: 优先应用
2. **高置信度 + 低支持度**: 特殊场景应用
3. **低置信度 + 高支持度**: 需要进一步验证
4. **低置信度 + 低支持度**: 暂不应用

#### 3.2 持续优化建议
- 📊 **定期重新分析**: 建议每月重新分析一次
- 🔄 **规则更新**: 根据新数据更新业务规则
- 📈 **效果跟踪**: 跟踪应用规则后的业务效果
- 🎯 **参数调优**: 根据业务需求调整分析参数

## 常见问题解答

### Q1: 分析结果不符合预期怎么办？
**A**: 
1. 检查数据质量和完整性
2. 调整分析参数 (置信度、支持度)
3. 扩大或缩小时间范围
4. 考虑数据的季节性和周期性因素

### Q2: 如何判断规则的业务价值？
**A**:
1. 结合业务场景评估规则的实用性
2. 考虑规则的可操作性和实施成本
3. 评估规则对业务目标的贡献度
4. 通过A/B测试验证规则效果

### Q3: 多久重新分析一次比较合适？
**A**:
- **高频变化业务**: 每周分析
- **中频变化业务**: 每月分析  
- **低频变化业务**: 每季度分析
- **重大变化后**: 立即重新分析

### Q4: 如何处理相互矛盾的规则？
**A**:
1. 检查规则的适用条件和范围
2. 分析规则的时效性
3. 考虑业务场景的差异
4. 通过专家判断确定优先级

### Q5: 预测准确率不高怎么办？
**A**:
1. 增加历史数据量
2. 考虑更多影响因素
