### 24.5 Token 过期

**实现位置**: `src/services/security_service.py` - `JWTManager.validate_token()`

**验证项目**:
- [x] Token 包含过期时间 (exp claim)
- [x] 默认过期时间为 1 小时
- [x] 过期 token 验证失败
- [x] 返回明确的过期错误信息
- [x] 过期时间可配置

**测试方法**:
```python
# 创建短期 token
token = jwt_manager.create_token(
    user_id="test_user",
    roles=["user"],
    expiry_seconds=1  # 1 秒后过期
)

# 等待过期
time.sleep(2)

# 验证失败
with pytest.raises(ValueError, match="Token has expired"):
    jwt_manager.validate_token(token)
```
