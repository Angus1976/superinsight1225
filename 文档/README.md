本目录包含 SuperInsight 项目的所有文档，按功能模块分类组织。

## 📁 目录结构

```
文档/
├── README.md                    # 本文件

├── 部署指南/                    # 部署和安装指南
├── Docker/                      # Docker 容器化相关
```

## 📚 各目录说明

### 快速开始
- **用途**: 新手入门
- **包含**: 快速启动指南、Docker 快速启动、登录指南等
- **推荐**: 首先阅读此目录

### 部署指南
- **用途**: 系统部署和安装
- **包含**: 部署说明、本地部署、测试指南等
- **推荐**: 部署前必读

### Docker
- **用途**: 容器化部署
- **包含**: Docker 配置、操作指南、诊断报告等
- **推荐**: 使用 Docker 部署时参考

### 登录系统
- **用途**: 认证和登录

- **推荐**: 登录问题时查看

### 前端开发
- **用途**: 前端开发和测试
- **包含**: 测试指南、验证报告、修复报告等
- **推荐**: 前端开发时参考

### 国际化翻译
- **用途**: 多语言支持

- **推荐**: 添加新语言时参考

- **推荐**: 了解功能实现时查看

- **包含**: 各个任务的完成报告
- **推荐**: 查看项目进度时参考

### 执行报告
- **用途**: 执行和部署记录

- **推荐**: 了解执行过程时查看

### 状态报告

- **推荐**: 检查系统状态时查看

### 问题修复
- **用途**: 问题解决
